# Java AIO 线程模型
Java AIO 与 NIO 的最大差别，在于前者不要求开发人员自己去设计线程模型。
Java 已将 AIO 接口设计的非常巧妙，线程模型与 I/O 模型的集成度也非常成熟，极大的简化了通信开发难度。
然而关于 AIO 线程模型的介绍资料却非常少，本文将通过官方提供的一些信息并结合自身对 JDK 源码的理解给大家作一个简要介绍。
