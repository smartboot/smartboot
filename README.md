# 关于smartboot

一个人的开源组织。

## 作品
- smart-socket   
    仅800多行代码实现极简、易用、高性能的java AIO通信框架，也许是IM、RPC、IoT开发的最佳选择
    
- smart-boot    
    基于springboot封装的微服务框架.

- somart-ioc    
    Java NIO实现的smart-socket android 版通信框架
    
- maven-mydalgen-plugin    
    mybatis代码自动生成的Maven插件，简化sql书写，避免大量动态sql。
